# Batch 报告

Batch 的测试结果、执行证据和验收结论保留在仓库根目录的 `reports/`，以便与运行产物和代码审阅
边界保持一致。MkDocs 的标准配置只能以 `docs/` 作为文档源，因此本地站点不复制该目录。
