# 报告索引

`reports/` 只存放测试结果、执行证据、验收结论与环境记录；设计、范围和验收要求位于 `docs/batches/`。按批次进入 `reports/batches/`，避免把报告当作当前项目状态。

| 批次 | 结论入口 |
|---|---|
| Batch 1 | `batches/batch-01/BATCH-1-RESULT.md` |
| Batch 2 / 2.5 | `batches/batch-02/`、`batches/batch-02-5/` |
| Batch 3A / 3B | `batches/batch-03a/`、`batches/batch-03b/` |
| Batch 4B-R1 / R2 | `batches/batch-04b-r1/`、`batches/batch-04b-r2/` |
| Batch 06 K/J paper | `batches/batch-06-kj-paper/RESULT.md` |
